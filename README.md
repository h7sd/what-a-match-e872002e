# What A Match

A social platform migrated from Vite + React to **Next.js 15** for v0 compatibility.

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser.

## Tech Stack

- **Next.js 15** - React framework with App Router
- **TypeScript** - Type safety
- **Tailwind CSS** - Styling
- **shadcn/ui** - UI components
- **Supabase** - Backend and authentication
- **Radix UI** - Headless UI primitives

## Project Structure

```
/app          - Next.js App Router pages
/components   - Reusable UI components
/lib          - Utility functions and helpers
```

## Migration Notes

This project was migrated from Vite + React SPA to Next.js 15 to support v0 development environment. The original Vite structure has been archived.
